v3.7.0.9 릴리즈 노트

ENHANCEMENT
Deeplearing
- BMP Training job이 deprecate되어 기존 BMP job에서 clone기능 제한
- training, evaluation용 채널의 repeat, shuffle 등 기본 입력값을 다르게 적용
- description, error message 강화
- job 삭제 시 기본 동작을 파일도 삭제하는 것으로 변경
MINOR FIXES
Engine
- Flow output 처리를 link에서 copy로 변경
- Spark global group by 의 Iterator 에 InterruptibleIterator 적용
- DataRoot path 설정 보완
- Script 실행 옵션 추가
Analytics App
- Publish 시 Snapshot On/Off 설정 기능 추가
- Single 템플릿에서 Condition상자 확장 기능 추가
- DatePicker 컨트롤 추가
- Secure Mode에서 계정 정보 연동할 수 있는 기능 추가
