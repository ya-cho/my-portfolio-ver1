DB 이전 가이드

  • DB 이전은 신청 항목에 관계 없이 최초 접수 1회에 한해서 지원하고 있습니다. (중복 신청을 하시거나 기존 미신청 항목에 대한 추가 신청은 반려됩니다)
  • 이전할 상품 페이지가 최소 10개 이상인 경우에만 이전이 가능합니다.
  • 상품 페이지의 주문 옵션 항목이 4개 이상인 상품은 데이터 이전이 불가하며, 스마트 스토어의 경우 특성상 상품 페이지 이전만 가능합니다.
  • DB 이전 서비스에서 디자인과 보유쿠폰은 포함되지 않습니다.
  • DB 이전 작업 중 기존 쇼핑몰에서 발생하는 회원가입, 상품 주문 건은 이전되지 않습니다.
  • 과거 주문 데이터는 DB 이전 후 정보 확인만 가능하며, 주문 상태 변경 등의 수정은 불가합니다.
  • 이전된 회원 계정의 비밀번호는 아이디 + 휴대폰 뒷자리(4자)로 초기화됩니다. (휴대폰 번호가 없는 회원은 db이전에서 제외됩니다.)
  • DB 이전 처리 결과는 완료 이후 문자로 발송되며, 신청 철회는 당사 홈페이지 게시판에 남겨주시기 바랍니다.
