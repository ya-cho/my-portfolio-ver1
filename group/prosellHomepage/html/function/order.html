로그인 회원가입

주문/배송관리

오배송과 실수를 없애는 시스템

모든 판매조건을 커버

다양한 상품을 각각의 조건과
방식으로 판매할 수 있어요.

보기 쉬운 주문서

주문서 하나에 다양한 정보를
탭으로 이동하고 확인 가능해요.

편리한 일괄처리

주문목록에서 한방에
배송 일괄처리가 가능해요.

내 입맛대로 엑셀 다운로드

설정한 목록 값 대로
마음 대로 다운로드 가능해요.

스마트 자동발주 시스템

벤더사 상품 주문 접수 시 발주서를 자동생성하고
지정된 시간에 이메일이 자동으로 발송돼요.
배송상태와 배송조회까지 한방에 해결!

필요했던 메모기능

고객 응대 내용을 모두 기억하기 쉽지 않죠?
어디서든 찾기 쉬운 메모 기능으로
메모하고 확인해보세요!
